# System Memory

## Architectural Shifts
- **Transition to Rust:** The project has pivoted from an uninitialized or submodule-heavy structure into a single Cargo workspace.
- **Consolidation Paradigm:** All development moving forward will focus on importing logic directly into the central Rust monolithic structure rather than managing disjointed repositories.

## Directives
- Commits must strictly track single, unified version numbers.
- Maintain high coverage in the UI with interactive forms.
- Comments must explain *why*, not *what*.
